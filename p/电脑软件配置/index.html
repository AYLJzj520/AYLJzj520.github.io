🚀 前言

每次重装操作系统，我们都希望能迅速恢复工作环境，减少不必要的配置时间。正如古语所说：“工欲善其事，必先利其器”，只有准备好合适的工具，才能高效地完成任务。

本文将为你提供一份完整的电脑配置指南，从系统基础设置到各类必备软件的安装，帮助你在重装系统后快速恢复工作环境，提升个人电脑的工作效率。

🔧 关闭Windows自动更新

🔧 关闭Windows Update服务

步骤：

  1. Win+R 输入 services.msc
  2. 在弹出的服务窗口中，找到 Windows Update 选项并双击打开
  3. 在弹出的属性对话框中，将 启动类型 设置为 禁用
  4. 单击 恢复 选项卡，将所有失败操作设置为 无操作
  5. 点击 应用 和 确定 保存设置
🔧 本地组策略编辑器

步骤：

  1. Win+R 输入 gpedit.msc
  2. 依次导航：计算机配置 > 管理模板 > Windows组件 > Windows更新
  3. 找到 配置自动更新 并双击打开，选择 已禁用，点击 确定
  4. 找到 删除使用所有Windows更新功能的访问权限，双击打开
  5. 选择 已启用，点击 确定 保存设置
🔧 任务计划程序

步骤：

  1. Win+R 输入 taskschd.msc
  2. 导航到路径：任务计划程序库/Microsoft/Windows/WindowsUpdate
  3. 选中 Schedule Start，在右侧操作面板中点击 禁用
🔧 关闭安全验证弹窗

永久关闭"无法验证发布者，你确定要运行此软件吗"弹窗

步骤：

  1. Win+R 输入 gpedit.msc 打开本地组策略编辑器
  2. 导航：用户配置 → 管理模板 → Windows组件 → 附件管理器
  3. 在右侧找到 中等风险文件类型的包含列表，设置为 已启用
  4. 在 指定中等风险扩展名 中输入：.cmd; .bat; .exe; .js
  5. 点击 确定，然后 Win+R 输入 gpupdate 更新策略
🔒 关闭BitLocker加密

方法一（推荐）：

  1. 开始菜单 → 设置 → 更新和安全 → 设备加密 → 点击 [关闭]

方法二：

  1. 控制面板或"此电脑"中找到要关闭加密的磁盘
  2. 右键点击 → 管理 BitLocker → 选择 [关闭 BitLocker]
